Stability

Set initial conditions

pars = data.frame(rprey=0.95, alpha=0.01, eff=0.6,pmort=0.4, K=2000, huntr = 0.2)

days = seq(from=1, to=500)

currpop = c(prey=90, pred=30)

Run ODE Solver

res = ode(func=lotvmodK, y=currpop, times=days, parms=pars)

res_df = as.data.frame(res) %>% gather(key="animal", value="pop",-time)

p1=ggplot(res_df, aes(time, pop, col=animal))+
  geom_line() +
  theme_minimal()

p1

We will define stability as each population being maximum 50% of the other population (cannot be more than double the other population) and both populations must still have members by the end of the time period (cant die out)
